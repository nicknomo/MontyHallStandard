# MontyHallStandard
This is the simulation for the Standard Monty Hall Scenario

This project was made as a comparison to the modified version of the Monty Hall problem.  It can be found here:

https://github.com/nicknomo/MontyHallModified/tree/main

A full writeup describing the purpose of this code and the modified one can be found in the other projects README, found here:

https://github.com/nicknomo/MontyHallModified/blob/main/README.md


# Results
Switching results in ~66% win rate

Not switching results in a 33% win rate

# Example Output

Number of games played: 900000

If contestant STAYS:

  Wins: 300111
  
  Losses: 599889
  
  Win rate: 33.35%
  

If contestant SWITCHES:

  Losses: 300111
